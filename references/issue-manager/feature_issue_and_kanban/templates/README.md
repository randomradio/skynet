# Issue 正文模板

各类型与仓库 Issue 模版对应，选择类型后预览与创建时会自动带出对应正文结构。

| 类型 | 文件名 | 说明 |
|------|--------|------|
| Doc Request | Doc_Request.md | 文档/PRD 类 |
| Customer Project | Customer_Project.md | 客户项目 |
| MO Feature / MO Bug | MO_Feature.md, MO_Bug.md | MO 侧功能/Bug |
| MOI Feature / MOI Bug / MOI SubTask | MOI_Feature.md, MOI_Bug.md, MOI_SubTask.md | MOI 侧 |
| Test Request | Test_Request.md | 测试需求 |
| EE Feature | EE_Feature.md | EE 功能 |
| User Bug | User_Bug.md | 用户反馈 Bug |

可直接编辑本目录下 `.md` 文件以调整各类型的默认正文结构；与仓库 `.github/ISSUE_TEMPLATE` 不一致时，以仓库为准或在此同步。
