## 📧 如何配置邮件发送

### 方法1: 使用环境变量（推荐）

```bash
# 设置SMTP用户名（通常是邮箱地址）
export SMTP_USER="your_email@example.com"

# 设置SMTP密码（QQ邮箱需要使用授权码，不是登录密码）
export SMTP_PASSWORD="your_smtp_password_or_auth_code"

# 设置发件人邮箱（可选，默认使用SMTP_USER）
export EMAIL_FROM="your_email@example.com"

# 持久化配置（macOS/Linux）
echo 'export SMTP_USER="your_email@example.com"' >> ~/.zshrc
echo 'export SMTP_PASSWORD="your_smtp_password"' >> ~/.zshrc
```

### 方法2: 直接修改配置文件

编辑 `config/config.py` 文件：

```python
SMTP_USER = "your_email@example.com"
SMTP_PASSWORD = "your_smtp_password_or_auth_code"
EMAIL_FROM = "your_email@example.com"
```

### 📝 常用邮箱SMTP配置

#### QQ邮箱
- **SMTP Host**: `smtp.qq.com`
- **SMTP Port**: `587` (TLS) 或 `465` (SSL)
- **授权码获取**: 登录QQ邮箱 → 设置 → 账户 → 开启SMTP服务 → 生成授权码

#### Gmail
- **SMTP Host**: `smtp.gmail.com`
- **SMTP Port**: `587` (TLS) 或 `465` (SSL)
- **需要**: 开启"允许不够安全的应用访问"

#### Outlook/163邮箱
- **Outlook**: `smtp-mail.outlook.com:587`
- **163邮箱**: `smtp.163.com:465` 或 `smtp.163.com:25`
