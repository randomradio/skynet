# MatrixOne 数据库连接测试使用说明

## 📋 概述

`test_matrixone_connection.py` 是一个全面的 MatrixOne 数据库连接测试脚本，用于验证数据库连接配置是否正确，以及数据库功能是否正常。

---

## 🚀 使用方法

### 1. 直接运行测试

```bash
cd "/Users/wupeng/Desktop/GitHub_Issue_智能管理系统"
python3 test_matrixone_connection.py
```

### 2. 测试内容

测试脚本会执行以下 10 项测试：

1. **连接字符串格式检查** ✅
   - 验证连接字符串格式是否正确
   - 检查是否包含所有必要组件

2. **创建数据库引擎** ✅
   - 测试 SQLAlchemy 引擎创建
   - 验证连接池配置

3. **基本连接测试** ✅
   - 测试是否能成功连接到数据库
   - 执行简单的 SELECT 查询

4. **数据库存在检查** ✅
   - 验证指定的数据库是否存在
   - 检查当前使用的数据库是否正确

5. **表存在检查** ✅
   - 检查必需的表是否存在：
     - `issues_snapshot`
     - `issue_relations`
     - `comments`

6. **表结构检查** ✅
   - 验证表结构是否完整
   - 检查关键字段是否存在

7. **CRUD 操作测试** ✅
   - 测试 INSERT（插入）
   - 测试 SELECT（查询）
   - 测试 UPDATE（更新）
   - 测试 DELETE（删除）

8. **连接池测试** ✅
   - 测试并发连接
   - 验证连接池功能

9. **数据库版本信息** ✅
   - 获取数据库版本
   - 获取当前用户和数据库信息

10. **事务测试** ✅
    - 测试事务回滚
    - 测试事务提交

## 📊 测试结果解读

### ✅ 所有测试通过

如果所有测试都通过，您会看到：

```
🎉 所有测试通过！MatrixOne 数据库连接正常，可以正常使用。
```

**这意味着：**
- ✅ 连接配置正确
- ✅ 数据库可访问
- ✅ 表结构完整
- ✅ 所有功能正常

### ⚠️ 部分测试失败

如果部分测试失败，脚本会显示详细的错误信息：

#### 常见问题及解决方案

**1. 连接超时 (Connection Timeout)**

```
错误: Can't connect to MySQL server on 'xxx' (timed out)
```

**可能原因：**
- 网络连接问题
- 数据库服务器不可访问
- 防火墙阻止连接
- VPN 未连接（如果数据库需要 VPN）

**解决方案：**
- 检查网络连接
- 确认数据库服务器地址和端口正确
- 检查防火墙设置
- 如果使用云数据库，确认是否允许当前 IP 访问

**2. 认证失败 (Authentication Failed)**

```
错误: Access denied for user 'xxx'@'xxx'
```

**可能原因：**
- 用户名或密码错误
- 用户没有访问权限

**解决方案：**
- 检查 `config/config.py` 中的用户名和密码
- 确认数据库用户有访问权限

**3. 数据库不存在**

```
错误: Unknown database 'xxx'
```

**解决方案：**
- 在 MatrixOne 中创建数据库：
  ```sql
  CREATE DATABASE github_issues;
  ```

**4. 表不存在**

```
错误: Table 'xxx' doesn't exist
```

**解决方案：**
- 运行主程序，表会自动创建
- 或手动运行 `modules/database_storage/mo_client.py`

## 🔧 配置检查

在运行测试前，请确认 `config/config.py` 中的配置正确：

```python
# MatrixOne 配置
MATRIXONE_HOST = "019c3caf-ae5a-7ced-ac5b-37d070f153f5.db.moi.matrixorigin.cn"
MATRIXONE_PORT = "6001"
MATRIXONE_USER = "admin"
MATRIXONE_PASSWORD = "Adminadmin1"
MATRIXONE_DATABASE = "github_issues"
```

**连接字符串格式：**
```
mysql+pymysql://admin:Adminadmin1@019c3caf-ae5a-7ced-ac5b-37d070f153f5.db.moi.matrixorigin.cn:6001/github_issues
```
