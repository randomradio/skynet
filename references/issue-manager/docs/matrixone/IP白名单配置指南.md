# MatrixOne Intelligence IP 白名单配置指南

## 📍 当前需要添加的IP地址
**公网IP**: `49.87.13.180`

## 🎯 配置步骤

### 方法一：通过 MatrixOne Intelligence 控制台（推荐）

#### 步骤 1: 登录控制台
1. 访问 MatrixOne Intelligence 控制台
   - 网址通常是：`https://intelligence.matrixorigin.cn` 或类似地址
   - 使用您的账号登录

#### 步骤 2: 找到您的实例
1. 在控制台首页，找到实例列表
2. 找到名为 **"实例_260208测试"** 的实例
3. 点击实例名称进入实例详情页

#### 步骤 3: 进入安全设置
1. 在实例详情页，找到 **"安全设置"** 或 **"网络设置"** 或 **"访问控制"** 标签
2. 可能的位置：
   - 左侧菜单中的 "安全" 或 "网络"
   - 顶部标签页中的 "安全设置"
   - 实例信息中的 "白名单" 或 "IP白名单"

#### 步骤 4: 添加IP白名单
1. 找到 **"IP白名单"** 或 **"访问IP列表"** 设置
2. 点击 **"添加"** 或 **"新增"** 按钮
3. 输入IP地址：`49.87.13.180`
4. 或者输入IP段（如果您的IP会变化）：`49.87.13.0/24`
5. 添加备注（可选）：`本地开发环境`
6. 点击 **"确定"** 或 **"保存"**

#### 步骤 5: 等待生效
- 通常配置会立即生效，但有时可能需要等待几分钟
- 配置完成后，可以重新测试连接

### 方法二：通过命令行工具（如果提供）

如果 MatrixOne Intelligence 提供了 CLI 工具，可以使用命令行配置：

```bash
# 示例命令（具体命令可能不同）
moctl whitelist add --instance "实例_260208测试" --ip "49.87.13.180"
```

### 方法三：通过 API（如果提供）

如果提供了 API，可以使用 API 配置：

```bash
curl -X POST "https://api.matrixorigin.cn/v1/instances/{instance_id}/whitelist" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "ip": "49.87.13.180",
    "description": "本地开发环境"
```

## ⚠️ 注意事项

1. **IP地址可能会变化**
   - 如果您的网络使用动态IP，IP地址可能会变化
   - 建议使用IP段（如 `49.87.13.0/24`）而不是单个IP
   - 或者定期检查并更新IP地址

2. **安全考虑**
   - 不要添加 `0.0.0.0/0`（允许所有IP），这会带来安全风险
   - 只添加您实际使用的IP地址

3. **配置生效时间**
   - 通常立即生效
   - 如果不行，等待1-2分钟后重试

4. **多个环境**
   - 如果有多个开发环境，需要分别添加每个环境的IP

## 🆘 如果仍然无法连接

如果配置IP白名单后仍然无法连接，请检查：

1. **确认IP地址正确**
   ```bash
   curl -s ifconfig.me
   ```
   确保显示的IP与添加的白名单IP一致

2. **检查端口号**
   - 确认控制台显示的端口号是否为 6001
   - 如果不是，更新 `config/config.py` 中的端口配置

3. **联系技术支持**
   - 如果以上步骤都正确但仍无法连接
   - 联系 MatrixOne 技术支持
   - 提供诊断报告：`连接诊断报告.md`
