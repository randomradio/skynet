# 依赖安装说明

## 快速安装

```bash
# 安装所有依赖
pip install -r requirements.txt
```

## 依赖包说明

### 必需依赖（MO数据库）

1. **sqlalchemy** (>=2.0.23)
   - 数据库ORM框架
   - 用于连接和操作MatrixOne数据库

2. **pymysql** (>=1.1.0)
   - MySQL协议驱动
   - MatrixOne使用MySQL协议，必需此驱动

### AI服务依赖

3. **anthropic** (>=0.18.0)
   - Claude AI服务SDK
   - 作为备用AI服务（当千问失败时使用）

4. **dashscope** (>=1.17.0)
   - 通义千问AI服务SDK
   - 优先使用的AI服务

### 其他依赖

5. **httpx** (>=0.25.2)
   - HTTP客户端库
   - 用于GitHub API请求

6. **python-dotenv** (>=1.0.0)
   - 环境变量管理
   - 用于读取配置文件

## 验证安装

运行以下命令验证所有依赖是否已安装：

```bash
python3 -c "import sqlalchemy, pymysql, httpx, anthropic, dashscope; print('✅ 所有依赖已安装')"
```

如果缺少某个依赖，会显示错误信息，例如：
```
ModuleNotFoundError: No module named 'dashscope'
```

## 单独安装缺失的依赖

如果某个依赖缺失，可以单独安装：

```bash
# 安装数据库依赖
pip install sqlalchemy pymysql

# 安装AI服务依赖
pip install anthropic dashscope

# 安装其他依赖
pip install httpx python-dotenv
```

## 常见问题

### 1. 安装dashscope失败

如果安装dashscope时遇到问题，可以尝试：

```bash
# 使用国内镜像
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple dashscope
```

### 2. 安装pymysql失败

如果安装pymysql失败，可能需要先安装编译工具：

```bash
# macOS
xcode-select --install

# 然后重新安装
pip install pymysql
```

### 3. 权限问题

如果遇到权限问题，可以使用：

```bash
pip install --user -r requirements.txt
```
