# @skynet/web

Next.js App Router application for the Skynet platform foundation.

## Commands

```bash
pnpm --filter @skynet/web dev
pnpm --filter @skynet/web lint
pnpm --filter @skynet/web typecheck
npm --prefix apps/web run test
```

## Included In Sprint 0

- Basic landing page and dashboard shell at `/dashboard`.
- Protected API sample route at `/api/example` using Bearer JWT verification.
